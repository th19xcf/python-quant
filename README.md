# 中国股市量化分析系统

基于Windows平台的Python量化分析系统，专注于中国股市的多维度分析与智能决策支持。

## 项目定位

整合多种数据源，提供强大的技术分析、宏观经济分析和情绪分析能力，结合主流量化框架，为投资者提供可视化的分析工具和自动化交易解决方案。

## 技术栈

- **基础环境**：Windows 10/11、Python 3.13+, 虚拟环境配置
- **数据源**：通达信数据文件、AkShare API、网络爬虫、财经新闻API
- **量化框架**：QLib、Valuecell、backtrader
- **UI框架**：PySide6、pyqtgraph、plotly
- **数据库**：MySQL 9.0+、Redis（可选）
- **数据处理**：Polars、pandas、numpy
- **技术分析**：ta
- **自然语言处理**：jieba、snownlp、transformers

## 核心功能

### 1. 数据获取与管理
- 通达信数据文件解析与导入
- AkShare API数据获取与更新
- 宏观经济数据爬取
- 财经新闻与社交媒体数据爬取
- 数据清洗、标准化与存储

### 2. 技术分析
- 内置常用技术指标（MA、MACD、KDJ、RSI等）
- 自定义公式系统
- 技术指标可视化
- 指标信号生成

### 3. 宏观经济分析
- 宏观经济数据可视化
- 经济指标与股市相关性分析
- 宏观经济周期判断
- 政策影响分析

### 4. 情绪分析
- 财经新闻情感分析
- 社交媒体情绪监测
- 市场情绪指数构建

### 5. 量化分析
- 多框架集成（QLib、Valuecell）
- 策略回测引擎
- 绩效评估指标
- 风险分析

### 6. 股票推荐
- 多因子选股模型
- 技术面选股
- 基本面选股
- 买卖时点信号生成

### 7. 自动交易（后期）
- 券商API对接
- 交易信号执行
- 仓位管理
- 风险控制

## 安装与使用

### 环境准备

1. 安装Python 3.13+（推荐使用Anaconda）
2. 创建虚拟环境：
   ```
   python -m venv venv
   venv\Scripts\activate
   ```
3. 安装依赖：
   ```
   # 使用pip
   pip install -r requirements.txt
   
   # 或使用poetry
   poetry install
   ```

### 配置

1. 复制配置文件模板并修改：
   ```
   cp config.example.toml config.toml
   ```
2. 配置数据库连接、API密钥等信息

### 运行

```
python -m src.main
```
